# 项目名称：基于统计学分布的逆势量化策略 (v1.0)
“在极端的斜率面前，平庸的逻辑会崩塌；唯有对风险的定价，才能换来生存。”

### 1. 策略逻辑 (The Logic)

信号识别： 当价格跌穿布林带下轨（2 Sigma 边缘事件）且 RSI < 25（极度恐慌）时入场。

风险对冲： 识别“高斜率”垂直下跌行情，通过金字塔式倍投 (Pyramid Scaling) 摊薄成本。

退出机制： 回归均线（MA20）或 RSI 超买即清仓。

### 2. 回测表现 (Backtest Results)

初始资金： 10,000 USD

最终收益： +2.8% (10,280.39 USD)

最大回撤： -5.3%

超额收益 (Alpha)： 相比 Buy & Hold 跑赢 29.87%。

### 3. 统计学思考 (Statistical Insights)

分析了 BTC 与科技股（NVDA/AAPL）的低相关性 (Correlation < 0.1)。

通过回测验证了：在非正态分布的“胖尾”行情下，仓位管理比入场点更重要。
